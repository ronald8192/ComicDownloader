# Comic Downloader

Downloader for cartoomad.com

## Download and Install dependencies
    git clone --depth 1 https://github.com/ronald8192/ComicDownloader.git
    cd ComicDownloader
    npm install

## How to use

```
npm start {comicId}
# or
node index.js {comicId}
```

File will save to `./download/comicId/`
 
## Tested in

    node 4.4.7
    npm  3.10.5
    Ubuntu Linux 14
